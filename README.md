# A typescript mini server for ZKWASM server side application

## Dependency

- Redis
```
sudo add-apt-repository ppa:redislabs/redis
sudo apt-get update
redis-server
```

- Mongodb
```
mongodb --dbpath db
```

- Node js
```
./ts >>> npm install
./ts >>> npm run server
```
